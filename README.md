# Camtracker_YOLOv8
Testing Project for Camera Tracking, based on YOLOv3 &amp; v8

This is a test project for my Video-Object Detector with YOLOv3 & V8. v3 is optimized for Images, while v8 is used for the live video feed.

for testing, you can download this repo and use this video as your video test:
https://youtu.be/MNn9qKG2UFI

This project requires you to set up your local virtual environment, you can set it up with python 3.10

you can also download the .WEIGHTS file of yolov3, from this link:
https://github.com/patrick013/Object-Detection---Yolov3/blob/master/model/yolov3.weights

Thank you!
